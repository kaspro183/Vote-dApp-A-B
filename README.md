# Vote dApp (A/B) – Base Sepolia

Vote unique par adresse sur **A** ou **B**, résultats publics on-chain.

## Déploiement (Remix, sans terminal)
1. Ouvre https://remix.ethereum.org
2. Colle le contrat depuis `contracts/`.
3. Compile en `^0.8.20`.
4. Dans MetaMask, sélectionne **Base Sepolia** (Chain ID `84532`).
5. Dans Remix: **Injected Provider - MetaMask** → **Deploy**.
6. Copie **l'adresse du contrat**.

## Utilisation de l'UI
- Ouvre `frontend/index.html` (Chrome + MetaMask).
- Colle l'adresse du contrat puis **Charger**.
- Utilise les boutons pour interagir.

## Upload sur GitHub (sans terminal)
- Nouveau repo → **Add file → Upload files**
- Glisse le **contenu** du dossier (pas le .zip), puis **Commit changes**.
